Planet Pocket Promotional Page

Planet Pocket is a simple educational PWA about planets. It was created using HTML, CSS, JavaScript, JSON, a web app manifest, and a service worker.

App Screenshots

How to Install

  1. Open the app using the web link or VS Code Live Server.
  2. Use Chrome or Edge for best install support.
  3. Click the Install App button or the install icon in the browser address bar.
  4. After installation, the app can open like a normal app and work offline.
