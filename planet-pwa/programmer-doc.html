How to Modify the JSON File

The file topics.json controls the app content. To add a new learning topic, add a new object inside the topics array.

{
  "title": "Neptune",
  "description": "Neptune is a cold blue planet far from the Sun.",
  "image": "assets/images/neptune.png",
  "audio": "assets/audio/neptune.mp3",
  "fact": "Neptune has strong winds.",
  "question": "Which planet is a cold blue planet?",
  "answer": "Neptune"
}

Steps to Add a New Topic

  1. Create a PNG image and put it in assets/images.
  2. Create an MP3 audio file using TTSMP3 or your own voice and put it in assets/audio.
  3. Add the topic information in topics.json.
  4. Add the new image/audio file names correctly.
  5. Refresh the app.

Main Files

  • index.html: main app layout
  • style.css: design
  • app.js: menu, quiz, audio, install button
  • topics.json: topic data
  • manifest.json: PWA install settings
  • service-worker.js: offline cache
