Planet Pocket PWA - Student Version

This project is an educational Progressive Web App about planets.

How to run in VS Code:
1. Unzip the folder.
2. Open the folder planet_pocket_student_pwa in VS Code.
3. Install the extension named Live Server.
4. In VS Code, right-click index.html.
5. Click Open with Live Server.

What should work:
- Planet menu appears on the left.
- Clicking a planet changes the lesson.
- PNG image changes.
- MP3 audio plays from assets/audio.
- Quiz question changes.
- Install button gives instructions or works in supported browsers.
- Promo, documentation, and programmer documentation pages open.

Audio note:
- This version uses MP3 files in assets/audio.
- To replace them with TTSMP3, go to ttsmp3.com, create audio, download each file, and name them:
  mercury.mp3, venus.mp3, earth.mp3, mars.mp3, jupiter.mp3, saturn.mp3
