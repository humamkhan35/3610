Documentation

App Features

  • Educational planet topics.
  • Menu is created dynamically from the JSON file.
  • Each topic has a title, description, PNG image, MP3 audio, fun fact, quiz question, and answer.
  • Users can play audio and answer a quick quiz.
  • The manifest file supports installing the app.
  • The service worker helps the app run offline.

How to Use

  1. Open index.html with VS Code Live Server.
  2. Click a planet in the menu.
  3. Read the information and play the MP3 audio.
  4. Answer the quiz question.

Files Included

The project includes HTML, CSS, JavaScript, JSON, PNG images, MP3 audio, manifest, service worker, and documentation pages.
